# 국문 학술 논문 작문 스타일 가이드

## 1. 문체 원칙
- **격식체**: ~이다, ~한다, ~된다 (논문 전체 일관)
- **3인칭 서술**: "본 연구는", "이는 ~을 시사한다"
- **과잉 수식 금지**: "혁명적", "획기적", "엄청난" 등 사용 불가 → 구체적 표현으로 대체

## 2. 학술 표현 패턴

### 서론 도입
- "~에 대한 학술적 관심이 증가하고 있다."
- "기존 연구는 ~에 집중해 왔으나, ~에 대한 논의는 부족하다."
- "본 연구의 목적은 ~을 제시하는 데 있다."

### 문헌 검토
- "~에 따르면(저자, 연도), ~이다."
- "~은 ~을 주장하였으며(저자, 연도), 이는 ~과 맥락을 같이 한다."

### 프레임워크 제안
- "본 연구는 ~을 구현하기 위한 N단계 아키텍처를 제안한다."
- "각 레이어는 ~의 기능을 수행하며, ~와 연결된다."

### 비교 분석
- "~과 ~을 비교한 결과, 다음과 같은 차이가 관찰된다."
- "구조적 수준에서 ~은 ~과 일대일 대응 관계를 형성한다."

### 결론
- "본 연구의 학술적 기여는 다음과 같이 요약된다."
- "향후 연구 과제로는 ~이 제안된다."

## 3. 인용 형식 (APA 7th 국문 적용)
- 본문 내: (이석민, 2026), (Lee, 2026)
- 2인: (홍길동·이석민, 2026)
- 3인 이상: (홍길동 외, 2026) / (Lee et al., 2026)
- 직접 인용: "인용문"(저자, 연도, p. 00)

## 4. 표·그림 표기
- 표: <표 1>, <표 2>
- 그림: <그림 1>, <그림 2>
- 캡션은 표 상단, 그림 하단에 위치

## 5. 피해야 할 표현
| 피하기 | 대안 |
|--------|------|
| 매우 중요한 | 핵심적인 / 필수적인 |
| 완벽하게 | 체계적으로 / 정합적으로 |
| 혁명적인 | 패러다임 전환적 / 구조적 전환을 수반하는 |
| 드라마틱한 | 유의미한 / 현저한 |
